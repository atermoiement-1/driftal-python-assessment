# Driftal Python + Integration Engineering Assessment

This repository contains my submission for the Driftal Python + Integration Engineering assessment.

## Files

- `assessment_solution.py` - Python solutions for the debugging, employee data processing, and API integration sections.
- `test_assessment_solution.py` - Unit tests using Python's built-in `unittest` module.
- `SUBMISSION.md` - Written answers for debugging notes, architecture, code review, and AI usage reflection.
- `requirements.txt` - Python dependency list.

## Run Tests

```bash
pip install -r requirements.txt
python -m unittest -v
```
